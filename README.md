# sunshine

A collection of small, useful, funny and elegant header-only C++ libraries.

## Dependencies

- Boost.Algorithm
- Boost.Filesystem
- Boost.Format
- Boost.TypeIndex
- Boost.TypeTraits

## Compile Dependencies

- CMake
- GCC 5+ or Clang 3.4+

## Build & Run

```sh
mkdir build && cd build
cmake ..
make && make tests
```
